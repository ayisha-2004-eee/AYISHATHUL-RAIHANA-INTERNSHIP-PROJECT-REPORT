Project Title: Automated Data Quality Assessment and Intelligent Imputation Pipeline
Description:This project focuses on developing a pipeline to automatically clean and preprocess raw datasets by handling missing values, detecting outliers, and normalizing features to improve data quality for analysis and machine learning.
What You Did:
>Loaded and explored the dataset
>Identified and analyzed missing values
>Performed basic imputation using median and mode
>Applied KNN imputation for improved accuracy
>Detected outliers using Isolation Forest
>Performed feature scaling using StandardScaler
>Compared basic and advanced preprocessing methods
>Evaluated data quality improvement using score and model performance
Tools Used:
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn
Jupyter Notebook
